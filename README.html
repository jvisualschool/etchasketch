🎨 Auto-Draw Etch-A-Sketch

맥북 최적화 디지털 Etch-A-Sketch - 4가지 테마 & 자동 그리기 & 트레이싱 모드

📖 프로젝트 소개

Auto-Draw Etch-A-Sketch는 클래식한 Etch-A-Sketch 장난감을 디지털로 재현한 웹 애플리케이션입니다. 실제 Etch-A-Sketch의 사실적인 디자인과 함께 현대적인 기능들을 추가하여, 이미지를 자동으로 그리거나 직접 따라 그릴 수 있는 두 가지 모드를 제공합니다.

🌟 핵심 특징

단일 파일 애플리케이션 - index.html 파일 하나만으로 완전히 동작하며, 별도의 서버나 라이브러리 없이 브라우저에서 바로 실행 가능합니다.

4가지 테마 모드 - Original, Light, Dark, Liquid Glass 테마로 본체와 버튼 색상까지 완전히 바뀌는 다이나믹한 경험을 제공합니다.

⚡ 주요 기능

🤖 자동 모드

이미지를 업로드하면 자동으로 경로를 분석하여 Etch-A-Sketch 스타일로 그려줍니다.

✏️ 트레이싱 모드

투명한 이미지를 배경으로 표시하여 직접 따라 그릴 수 있습니다. 실제 노브가 회전합니다!

🎨 4가지 테마 모드

🎨 Original

클래식한 빨간색 Etch-A-Sketch. 전통적인 디자인을 그대로 재현했습니다.

☀️ Light Mode

밝은 파스텔 핑크 테마. 부드럽고 따뜻한 색감으로 화사한 분위기를 연출합니다.

🌙 Dark Mode

어두운 사이버펑크 스타일. 파란색/보라색 계열로 밤에 사용하기 좋습니다.

💧 Liquid Glass

투명한 글래스 효과. 블러 배경과 반투명 본체로 미래적인 느낌을 제공합니다.

🎯 라인 굵기 조절

가늘게, 보통, 굵게 - 3단계로 라인 두께를 조절할 수 있습니다.

🌊 지그재그 폭 조절

깔끔, 보통, 거칠게 - Etch-A-Sketch 특유의 지그재그 효과를 3단계로 조절합니다.

🎛️ 실시간 노브 회전

트레이싱 모드에서 L/R 노브가 실제 Etch-A-Sketch처럼 회전합니다.

🎨 4가지 테마 모드

Original, Light, Dark, Liquid Glass - 본체와 버튼 색상까지 완전히 바뀝니다.

📱 멀티 플랫폼 지원

데스크톱, 모바일, 태블릿에서 모두 사용 가능하며 터치 입력을 지원합니다.

🎨 투명도 조절

트레이싱 모드에서 배경 이미지의 투명도를 10%~80%까지 조절할 수 있습니다.

⚡ 속도 조절

자동 그리기 속도를 1x~10x까지 조절하여 원하는 속도로 감상할 수 있습니다.

🎮 조작 방법

🖱️ 마우스/터치 조작

마우스 드래그 트레이싱 모드에서 자유롭게 그리기
터치 드래그 모바일에서 손가락으로 그리기
화면 흔들기 마우스 드래그로 화면 지우기 (실제 Etch-A-Sketch처럼!)

🎨 테마 변경

상단 중앙 테마 스위처 4가지 테마 모드 실시간 변경
자동 저장 선택한 테마가 자동으로 저장되어 다음 방문 시 유지

⌨️ 키보드 단축키

방향키 (↑↓←→) 트레이싱 모드에서 정밀한 픽셀 단위 이동
스페이스바 자동 그리기 일시정지/재생
R 키 화면 초기화
⌘+O (Ctrl+O) 이미지 파일 열기
⌘+S (Ctrl+S) 그린 이미지 저장
숫자키 (1-9) 자동 그리기 속도 조절

📋 사용법

🎨 테마 선택

테마 스위처 사용

화면 상단 중앙의 테마 스위처에서 원하는 테마를 선택합니다.

실시간 변경

테마를 선택하면 배경, 본체, 버튼 색상이 즉시 변경됩니다.

자동 저장

선택한 테마가 자동으로 저장되어 다음에 접속할 때도 유지됩니다.

🤖 자동 모드 사용법

이미지 업로드

"이미지 업로드" 버튼을 클릭하여 그리고 싶은 이미지를 선택합니다.

설정 조절

라인 굵기, 지그재그 폭, 그리기 속도를 원하는 대로 조절합니다.

자동 그리기 시작

"자동 그리기" 버튼을 클릭하면 이미지가 Etch-A-Sketch 스타일로 자동으로 그려집니다.

✏️ 트레이싱 모드 사용법

트레이싱 모드 선택

상단의 "트레이싱" 버튼을 클릭하여 트레이싱 모드로 전환합니다.

이미지 업로드

따라 그리고 싶은 이미지를 업로드합니다.

투명도 조절

배경 이미지의 투명도를 조절하여 보기 편한 상태로 맞춥니다.

트레이싱 시작

"트레이싱 시작" 버튼을 클릭한 후 마우스, 터치, 또는 방향키로 이미지를 따라 그립니다.

🔧 기술 사양

  • 파일 형태 단일 HTML 파일 (index.html)
  • 외부 의존성 없음 (완전 독립형)
  • 지원 브라우저 Chrome, Firefox, Safari, Edge (최신 버전)
  • 모바일 지원 iOS Safari, Android Chrome
  • 반응형 디자인 데스크톱, 태블릿, 모바일 최적화
  • 오프라인 동작 인터넷 연결 불필요
  • 이미지 형식 JPG, PNG, GIF, WebP
  • 테마 모드 4가지 (Original, Light, Dark, Liquid Glass)
  • 최적화 맥북 및 고해상도 디스플레이
